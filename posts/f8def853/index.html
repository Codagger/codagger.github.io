new-delete 与 malloc-free

new&delete 与 malloc&free 的比较

  • malloc/free 是 C 语言的库函数
  • new/delete new[]/delete[] 是 C++ 中的运算符
  • malloc/free 只是申请内存空间，释放空间
  • new/delete 不仅会申请内存空间， 还会根据类型初始化内存空间，this 指针绑定， 调用构造函数和析构函数进行初始化或者清理等操作
